# Public Content Audit

This repository is built from a private source archive. The public asset set is intentionally small and explicit.

## Approved assets

- Portrait image for homepage and social preview
- Current CV PDF
- One GUIDE framework figure
- One MatToolBench system figure

## Approved text sources

- English rewrites derived from the current resume
- Public-facing project summaries based on presentation materials
- Publication metadata that has been checked for venue-safe status display

## Explicit exclusions

- Review-preparation notes
- Experimental notebooks
- Unused paper drafts and supplemental assets
- Unpublished project material

## Ongoing rule

Before adding a new file, verify that it is suitable for a public repository and not merely suitable for a personal archive.
